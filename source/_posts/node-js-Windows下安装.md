# 首先去Node.js中文网下载安装包

- 偶数位为稳定版本，奇数位为非稳定版本（开发版）
- 稳定版本中已发布的API是不会改变的
- 奇数开头的开发版就是会不断变化更新

<!--more-->

[点这里去下载](http://nodejs.cn/download/)
