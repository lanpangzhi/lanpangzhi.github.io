node.js处理前台get请求

下面代码实现了用node.js接收前台发送的get请求。用到了http模块和url模块。

服务端代码

先搭一个服务器监听8000端口，再把请求的链接用URL模块解析成对象，判断请求路径，返回user。

客户端代码

在命令行执行 node http.js。
打开index.html 输入数据，点击提交按钮。
node输出 { user: ‘123’, pass: ‘abc’ } 123和abc 就是你输入的数据。
前台页面会显示你输入的user。
