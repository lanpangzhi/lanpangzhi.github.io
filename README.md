## 基于hexo 搭建的静态资源博客 主题选用 indigo 备用 next
```
git clone git@github.com:lanpangzhi/lanpangzhi.github.io.git
npm install hexo-cli -g
cd lanpangzhi.github.io
npm i
git submodule init
git submodule update
```
